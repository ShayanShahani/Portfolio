# 🍎 ShayanOS

A modern, interactive portfolio experience inspired by macOS, built to showcase my projects, skills, and work through a familiar desktop environment.

## ✨ Highlights

- 🖥️ Desktop-inspired user experience
- 🚀 Interactive applications and workflows
- 🎨 Custom themes, wallpapers, and animations
- 🔍 Spotlight-style search
- 💻 Interactive terminal experience
- 📱 Responsive design across devices
- ⚡ Fast, modern, and optimized

## 🛠️ Tech Stack

### Frontend

- Next.js
- React
- TypeScript
- Tailwind CSS

### UI & Components

- shadcn/ui
- Radix UI
- Lucide Icons

### Tooling

- Node.js
- npm
