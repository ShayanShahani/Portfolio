# 🍎 Shayan Shahani Portfolio

A modern, interactive personal portfolio built with a macOS-inspired desktop experience for large screens and a clean, dedicated mobile experience for phones and tablets.

This project is designed to present my work, skills, personality, and technical interests through a polished interface that feels more like a digital workspace than a traditional portfolio page.

## ✨ Highlights

- 🖥️ macOS-inspired desktop interface for large screens
- 📱 Dedicated mobile and tablet portfolio layout
- 🪟 Draggable, resizable, and animated application windows
- 🚀 Custom portfolio applications and interactive workflows
- 🎨 Light/dark mode, custom themes, wallpapers, and animations
- 🔍 Spotlight-style search experience
- ⚙️ macOS-style Settings, Control Center, Menubar, Dock, and system UI
- 💻 Interactive terminal-inspired experience
- 🌐 Safari-style portfolio browsing
- 📷 FaceTime-style camera preview app
- 🧩 Multiple custom apps, UI improvements, and bug fixes
- ⚡ Built with modern, optimized web technologies

## 🧠 About the Project

This is my personal portfolio, designed as an interactive digital workspace.

On desktop-sized screens, the portfolio becomes a macOS-inspired environment with a Dock, Menubar, draggable windows, app previews, system-style interactions, and custom applications.

On phones and tablets, the project switches to a completely separate mobile portfolio design focused on readability, simplicity, and a clean user experience.

The goal of this project is not only to show my work, but also to demonstrate attention to detail, UI/UX thinking, frontend architecture, interactivity, and practical problem-solving.
